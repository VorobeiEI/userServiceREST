Приложение работает в двух режимах:
1)Базовая аунтефикация пользователей по логину и паролю. Страница регистрации доступна без аунтефикации
Для запуска в таком режиме устанавливае профиль "DEV" - в Edit Run Configuration меню в поле Active Profiles.

2)Аунтефикация пользователей используя токены. Запускаем приложение юез указания профиля.
Для получения токена для зарегистрированого пользователя передаем его логин и праоль в параметрах запроса

Н=р: localhost:8080/oauth/token?grant_type=password&username=Jack&password=11111
В данном случае пользователь Jack должен быть зарегистрирован.

Для Выдачи токена в поля базовая аутентификация указываем юзера имеющего права выдавать токены(он реализован inmemory)

Username : trusted_client
password : secret

Получив токен имеем доступ к другим ресурсам АПИ.

Пример запроса с токеном
localhost:8080/user/getUsers/?access_token=a8d253a2-915b-46df-9f45-7ed425273b5c&page=0&size=10

Приложение использует Inmemory DB hsqldb
